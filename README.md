> This is awesome application


## Install

```sh
$ npm install --save sharyn-cli
```


## Usage

### JavaScript

```js
var sharyn-cli = require('sharyn-cli');

sharyn-cli('Rainbow');
```

### Command Line reference

```sh
$ npm install --global sharyn-cli
$ sharyn-cli --help
```

## Run Test
```sh
npm test
```
